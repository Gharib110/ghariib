# ZDI-25-125 0Day Ashlar-Vellum Cobalt XE File Parsing Integer Overflow Remote Code Execution Vulnerability

<br/>

<br/>
This vulnerability allows remote attackers to execute arbitrary code on affected installations of Ashlar-Vellum Cobalt. User interaction is required to exploit this vulnerability in that the target must visit a malicious page or open a malicious file. The ZDI has assigned a CVSS rating of 7.8. The following CVEs are assigned: CVE-2025-2021.
